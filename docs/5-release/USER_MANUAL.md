# User Manual (Panduan Penggunaan)

Buku panduan singkat ini menjelaskan cara menggunakan fitur-fitur utama di Aplikasi CBT Enterprise.

## Untuk Guru / Admin

### 1. Manajemen Mata Pelajaran & Bank Soal
1.  Login ke aplikasi dengan akun Guru/Admin.
2.  Buka menu **Mata Pelajaran** untuk menambah mata pelajaran baru.
3.  Buka menu **Bank Soal**, buat bank soal baru dan pilih mata pelajarannya.
4.  Klik nama Bank Soal untuk masuk ke detailnya.

### 2. Membuat & Mengimpor Soal
Ada dua cara menambahkan soal ke Bank Soal:
*   **Manual:** Klik tombol **Add Question**. Gunakan editor teks untuk mengetik soal, menambahkan opsi, dan menentukan jawaban benar. Anda juga dapat melampirkan URL media (Gambar/Audio/Video).
*   **Import Word:** Klik **Import from Word**. Pastikan file `.docx` Anda menggunakan format `SQ` (Mulai Soal) dan `EQ` (Akhir Soal). Sistem otomatis membaca opsi A, B, C, D dan `ANS: X` untuk jawaban benar.

### 3. Menjadwalkan Ujian
1.  Buka menu **Exams**, klik **Schedule Exam**.
2.  Isi judul, pilih mata pelajaran, dan tentukan rentang waktu serta durasi ujian (menit).
3.  Pilih soal-soal yang ingin dimasukkan dari Bank Soal yang telah dibuat.
4.  Bagikan "Token" ujian kepada siswa.

### 4. Mengawasi Ujian (Live Proctoring)
1.  Saat ujian berlangsung, buka menu **Monitoring**.
2.  Klik ujian yang sedang berjalan.
3.  Anda akan melihat daftar siswa yang sedang aktif (LIVE).
4.  Jika siswa mencoba membuka tab lain atau meminimize aplikasi, sebuah notifikasi merah akan muncul di kolom "Live Logs" beserta penambahan angka peringatan di kartu siswa.

### 5. Melihat Hasil & Memberi Nilai
1.  Buka menu **Exams**, klik icon dokumen ("View Results") di sebelah kanan ujian.
2.  Anda akan melihat daftar siswa yang telah selesai beserta nilai sementara.
3.  Untuk menilai soal Essay, klik **Details & Grade** pada salah satu siswa.
4.  Cari soal Essay, baca jawaban siswa, lalu ketikkan nilai di kotak "Points" dan klik icon simpan (Save). Nilai Total akan otomatis terupdate.
5.  Klik **Export Excel** di kanan atas untuk mengunduh rekap nilai seluruh siswa ke file `.xlsx`.

## Untuk Siswa

### 1. Memulai Ujian
1.  Login dengan Username dan Password yang diberikan sekolah.
2.  Di halaman utama (Dashboard), Anda akan melihat daftar Ujian yang Tersedia.
3.  Klik ujian, masukkan Token (jika diminta), dan klik Mulai.

### 2. Mengerjakan Ujian
1.  Waktu akan mulai berjalan mundur di kanan atas.
2.  Pilih jawaban yang benar, atau ketik jawaban untuk soal essay.
3.  Sistem secara otomatis menyimpan jawaban Anda saat Anda berpindah soal.
4.  **Peringatan:** Jangan berpindah tab atau membuka aplikasi lain! Sistem akan mencatat pelanggaran tersebut dan mengirimkannya langsung ke Guru pengawas.
5.  Jika sudah yakin, klik tombol **Selesai Ujian** warna merah di kanan atas.
